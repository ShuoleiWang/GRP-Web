How to Download

  • Click the "Download Software" button and select the download directoey.
  • Unzip the downloaded zip file, then right click the "ElementAnalysis.exe" file and choose "Run as Administrator" option to launch the software.
  • The software will be opened as below.

    • Software Description

    This software is for analyzing food elements properities.

    You should import food element data into the software through csv files. Then, you could check the data in software and choose whether generate an analysis report which contains analysis and statistical results.

    Software Usage

  • First, you should click the "Open File" button in the top-left corner in software interface.

  • Second, select the csv file you want to import and click "OK".

  • And input the name you want for your data.

  • The software interface should look like the image below.

  • If you want to generate an analysis report, you should click this button in the top-left corner in software interface.

  • Then, you chould select the options you want the report has.

  • Finally, click "OK" and select the path to save the generated report.

  • The report will be generated in your selected path.
